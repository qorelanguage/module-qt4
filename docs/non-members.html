Related Non-Members - QGlobalSpace

Qt4 contains some functions called “Related Non-Members” mentioned in the documentation. These functions are not part of default namespace or class's namespace. All these functions are used as static method of class QGlobalSpace.

Example:

QGlobalSpace::qRed(rgb)
