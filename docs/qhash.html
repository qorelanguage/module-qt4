QHash

Qt4 QHash<T1,T2> template class is not implemented in this module. Structures are converted to the Qore hash structure with all its features like keys are strings etc.

This part of Qore Qt binding is not finished. Please report any of missing conversions (your script will raise an exception with warning and additional informations.
